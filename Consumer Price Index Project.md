Project Overview
This project focuses on analyzing the Consumer Price Index (CPI) using Microsoft Excel to understand inflation trends and price changes over time. The objective is to transform raw data into meaningful insights that help track the cost of living and economic conditions.

📂 Dataset Description
The dataset contains CPI values categorized by time (monthly/yearly) and possibly by product categories or regions. It represents how the prices of goods and services change over time.

Identification of periods with high and low CPI growth
Comparative analysis across categories/regions
Insights into economic patterns affecting price changes

This project demonstrates how Excel can be effectively used for data analysis and visualization. It provides valuable insights into inflation trends and helps in understanding economic indicators like CPI.
